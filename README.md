# Public Agents Repository

This repository contains a collection of AI agents with their configurations, reports, knowledge bases, and workflows.

## Structure

Each directory represents a different agent with the following structure:
- `reports/` - Pre-configured report templates
- `knowledge/` - Knowledge base files and documentation
- `datasources/` - Data source configurations and connectors
- `workflows/` - Automated workflow definitions

## Available Agents

- **gcp-billing-agent** - GCP billing and cost optimization agent

## Getting Started

Each agent directory contains its own README with specific setup instructions and usage guidelines.
